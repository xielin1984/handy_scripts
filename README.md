# A collection of handy scripts used to post-process DFT and MD data
1. get_3d_prob.py: Map the positions of selected types/IDs of atoms in a supercell back to a unitcell. Helpful in getting the probability density P(r) of certain atoms.

Example usage: ovitos get_3d_prob.py traj.xyz

Need to specify the affine matrix relation between the supercell and the unitcell, types/IDs of atoms and nbins in three dimensions.

2. write_density_cube.py: Read the probability density generated by get_3d_prob.py and output a 3D volumetric data in cube format. Also need a <structure> file to specify the unitcell used to generate the .cube file.

Example usage: python write_density_cube.py --npz unitcell_probability_density_3d.npz --structure unitcell.xyz --output prob_density.cube
